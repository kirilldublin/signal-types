# Signal Types Classification — кластеризация сигналов сцинтилляционного детектора

Решение Kaggle-соревнования: https://www.kaggle.com/competitions/signal-types-classification

Задача — кластеризовать 23 479 сигналов органического сцинтилляционного
детектора (паратерфенил) на три группы:

- **кластер 0 / 1** — два типа частиц (гамма-кванты и нейтроны),
- **кластер 2** — аномальные / неклассифицируемые сигналы.

Решается методами обучения без учителя (clustering) с физической
интерпретацией результата.

## Что где смотреть

- [`solution.ipynb`](solution.ipynb) — итоговое эталонное решение, последовательно:
  1. загрузка и разбор формата данных;
  2. EDA — базовая линия, форма импульса, банана-плот, PSD-дискриминатор;
  3. feature engineering — заряды в коротком/длинном окнах, PSD-параметры,
     геометрия импульса, флаги качества;
  4. отбор и анализ признаков (в т.ч. корреляция с метаданными детектора);
  5. сравнение алгоритмов (KMeans, GMM, Agglomerative, DBSCAN, Bayesian GMM)
     и подбор гиперпараметров;
  6. финальная модель — гибрид правил + GMM(2);
  7. проверка устойчивости, формирование сабмита, выводы.

## Воспроизведение

```bash
pip install numpy pandas matplotlib scikit-learn
```

Данные `Run200_Wave_0_1.txt` берутся со страницы соревнования на Kaggle. Положите файл рядом
с ноутбуком и при необходимости поправьте `DATA_PATH` в начале раздела «Загрузка данных», после чего выполните все ячейки сверху вниз.

Результат детерминирован: все модели запускаются с фиксированным `random_state = 42`.
